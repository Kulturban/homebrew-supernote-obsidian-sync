# Kulturban Supernote-obsidian-sync

## How do I install these formulae?

`brew install kulturban/supernote-obsidian-sync/<formula>`

Or `brew tap kulturban/supernote-obsidian-sync` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "kulturban/supernote-obsidian-sync"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
